# Xyaa
A GUI based TextEditor.
 
### Features

 - Open
 - Save
 - Exit
 - Copy
 - Paste
 - Clear
 - ClearAll
 - Insert Date
 - Insert Line

 - Background Color 
 - Font Styles 
   - Italics, Bold, Normal, Underline
